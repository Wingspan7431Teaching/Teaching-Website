Exercise 5A

In your code, you will have the following array:

    Movements = [“movefoward”, “turnRight”, “skip”, “moveForward”, “moveRight”, “moveBack”]

Use a loop to loop through each item in the array and execute the action in the string. This will require multiple conditional statements (if & else if statements in this case) to check what content of the string is and call the corresponding method.

Assume any string that begins with “move” to be 1 meter and any string that begins with “turn” to be PI/2 radians.
