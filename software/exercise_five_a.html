Exercise 5A

Copy and paste the following array into your code.

    Movements = [“movefoward”, “turnRight”, “skip”, “moveForward”, “moveRight”, “moveBack”]

Task: Use a loop to iterate through each item in the array and execute the action in the string. This will require multiple conditional statements (if & else if statements in this case) to check what content of the string is and call the corresponding method.

Assume any string that begins with “move” to be 1 meter and any string that begins with “turn” to be PI/2 radians.
