Exercise Fifteen

In your code from the previous exercises (13 & 14), add a command to turn the motor to a specific angle using PID control.

Specifications/ Tips:

  • Make the command name descriptive for the task
  • Because the encoder for this motor is relative, you’ll have to wrap the input to the PID
  • To tune the PID you’ll have to watch how the motor spins, the values won’t be perfect.
  • The constructor for the command should take in the angle as a parameter.
