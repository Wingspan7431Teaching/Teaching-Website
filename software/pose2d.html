Pose2D Class

One frequently used class in FRC is the Pose2D class. The class stores information about a position on the field. Specifically, the class stores the x position, y position, and the rotation for a given point which can be accessed through getter methods.

The rotation in the Pose2D class is stored using another class called Rotation2D. The Rotation2D class stores an angle. The class has methods such as getRadians(), getDegrees(), and getRotations() which is incredibly useful when different classes use different units to measure angles.

An example is put below which creates a Rotation2D and then uses it to create a Pose2D:

    Rotation2D angle = Rotation2D.fromDegrees(180);

    Pose2D pose = new Pose2D(1.2,2.5,angle);

I’d highly recommend reading the details of the Pose2D and Rotation2D classes on the API docs. It’s good for both becoming more familiar with the API docs—which are a dense but useful resource— and understanding just how many different functionalities each of these classes offer.
