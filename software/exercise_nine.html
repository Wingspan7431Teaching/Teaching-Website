Exercise Nine

Task: Create an enum, Position, with three possible values: LEFT, RIGHT, and CENTER. Next, create a variable of type Position that will be any one of these values. Using a switch statement, run a specific routine based on the value of the Position variable.

For example, if the position is LEFT, the robot might drive forward and turn left. If it's CENTER, maybe it just drives forward. If it's RIGHT, it might drive forward and turn right.
