Exercise Four

The scratchbot methods needed for this exercise are:

  • Scratchbot.sb_getControllerLeftX()
  • Scratchbot.sb_getControllerLeftY()
  • Scratchbot.sb_setRobotMovement(xVelocity, yVelocity)

In order to make the Scratchbot.sb_setRobotMovement work, we need lambda expressions: () -> {}

You don’t have to worry about what that is just yet, copy and paste the setup below into your code. Then, replace each “?” with either X or Y. Note that zero, one, or both of the controller values may need to be multiplied by -1.

    Scratchbot.sb_setRobotMovement(
    () -> {return Scratchbot.sb_getControllerLeft?()},
    ()-> {return Scratchbot.sb_getControllerLeft?()}
    );

Figure out what axis to use for each parameter. When you’re done, write brief comments to explain the code.
