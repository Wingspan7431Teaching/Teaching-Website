Final Words

Welcome to the end of the lessons! Yay! If you’ve made it this far you should have a foundational knowledge of most concepts in java and FRC. There is, of course, much more to learn still. In the additional resources section of the lessons, there will be a variety of conceptual level lessons that teach various FRC concepts. But, now, you know the basics which means you can start doing projects for your team without a specific curriculum.

One of the biggest skills you’ll need for doing projects is the ability to use the WPILib docs and various API docs. A good programmer isn’t someone that has everything memorized, but someone that can quickly find what they need.

The WPILib docs are not entirely exhaustive, but they do present the vast majority of information on how to use WPILib. This is great because it means whatever you need you can usually find it in the docs! However, it’s also easy to get lost learning things that aren’t necessarily applicable to your task. Don’t be discouraged if it’s hard to learn through these documents at first, overtime you’ll become better at utilizing them.

Additionally, don’t forget about API docs. API docs give you descriptions of all the methods, variables, and features of classes within a library. WPILib and REVLib both have API documentation that is incredibly useful.

The other major hurdle you’ll find as you begin making more projects is problem solving. There will be times in which your program needs to have certain functionality that isn’t built into a class or your program has some strange bug. These issues may have solutions on Chief Delphi (which you should check out if you haven’t heard of it yet), but oftentimes you’ll need to come up with an algorithm or spend days debugging. This is, in my opinion, one of the most fun and rewarding parts of programming. It’s difficult and frustrating to find that solution, but it’s so worth it when you see it working as intended. If you get stuck, asking your team members, other teams, or people in online communities is a great way to learn.

Keep your head up, it’s impressive how far you’ve made it already. Best of luck! I can’t wait to see how you spread your wings!
