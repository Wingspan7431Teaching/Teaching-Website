Exercise Zero

You can find the template code for this lesson here. The process for getting the code on to your computer will be:

  1. Hit the fork button (top right)
  2. Hit create fork
  3. Open GitHub Desktop on your computer
  4. Click the button on the top right that says "Current repository"
  5. In the sidebar that opens, hit add, then clone
  6. Under your repositories, select the repository named exercise-zero (or something else if you renamed it during step 2).
  7. The local path will appear at the bottom, make sure you know that path
  8. Hit clone, If it prompts you anything else following this hit "For My Own Purposes"
  9. Congrats! The code is now on your computer
  10. Open wpilib vscode and open the folder with that path you looked at earlier.

The instructions are:

Task: Create a variable called speed with a value of 12.5, and a variable called timeSpentMoving with a value of 6. We’ll assume 12.5 is in meters/second and time is in seconds. Your program should multiply the two variables to get the distance traveled and print it in the console. The console output should be:

Distance Traveled: 75.0 meters

Feel free to change the values around afterwards to see what happens.
