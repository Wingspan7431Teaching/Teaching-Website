Implementing Controllers in Code

The motor controllers we usually use, SparkFlex and SparkMax, have built in PID controllers. We use these controllers to calculate our motor input instead of calculating it on the RIO. However, you can calculate it on the RIO with the PIDController class. You can access the PID in a SparkFlex/ SparkMax in a similar way to how you get encoders:

    //make a sparkMax motor

    SparkMax motorController = new SparkMax(0, MotorType.kBrushless);

    //make a PID controller object, the same format is used for SparkFlex

    SparkClosedLoopController PIDController =

     motorController.getClosedLoopController();

In order to edit the properties of the controller, like the coefficient values (KP, Ki, and KD), you can alter it in the config for the motor controller before making the object:

    //make the motor configuration

    SparkMaxConfig config = new SparkMaxConfig();

    //Access the closed loop controller’s settings

    config.ClosedLoop

The following list shows a few settings you’ll likely need to set:

.pid(double Kp, double Ki, double Kd)
Set the PID coefficients

.feedbackSensor(FeedbackSensor sensor)
This determines the sensor that will be used to get the error. The value for sensor may be:

  • FeedbackSensor.kAbsoluteEncoder: The encoder from .getAbsoluteEncoder()
  • FeedbackSensor.kPrimaryEncoder: The encoder from .getEncoder()
  • FeedbackSensor.kAlternateOrExternalEncoder: The encoder from .getExternalEncoder()
  • FeedbackSensor.kAnalogSensor
  • FeedbackSensor.kNoSensor

.positionWrappingEnabled(boolean enabled)
This “wraps” the value of the position between two values. For example, when we hit 2 radians of rotation on a motor we want the value to be considered 0 radians, so if we set wrapping to true we can make sure that the value wraps.

.positionWrappingInputRange(double minInput, double maxInput)
This defines the points at which the motor should wrap. For wrapping at 2 radians, minInput would be 0 and maxInput would be 2.

.outputRange(double minOutput, double maxOutput)
Sets the output range that the PID can send to the motor. The values for the parameters must follow: -1 ≤ minOutput ≤ 1 and -1 ≤ maxOutput ≤ 1

To configure feedforward control:

    config.ClosedLoop.feedForward

     .kV(val);

kV(val) sets the kV coefficient for the feedforward controller to val. Many other values can be set in this same manner including kG.

Final Example:

    SparkMax motorController = new SparkMax(0, MotorType.kBrushless);

    SparkMaxConfig config = new SparkMaxConfig();

    config.ClosedLoop

     .feedbackSensor(FeedbackSensor.kAbsoluteEncoder)

     .positionWrappingEnabled(true)

     .positionWrappingInputRange(0, 2*Math.PI)

     .outputRange(-1, 1)

     .pid(0.1, 0, 0.01);

    config.ClosedLoop.feedForward

     .kV(0.05);

    motorController.configure(config, true, true);

    SparkClosedLoopController PIDController = motorController.getClosedLoopController();
