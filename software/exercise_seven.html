Exercise Seven

Create a class called “MockMotor” in a separate file. This class should have the following:

  • A port variable (with a getter method)
  • A speed variable (with getter and setter methods)
  • A constructor

Normally, this class would also send the speed to the motor as input, but we'll handle that in future exercises.

Once you’re done with the specifications above, change the setter a bit to clamp the speed. This means that the speed variable will only be set if the input value is between -2 and 2.
