Exercise Eight

From here on out you will no longer be using the Lab file. Instead, you’ll make your own files in the Lab folder and import them into the Robot.java file where you can run the code.

When running Scratchbot methods outside of the runLab method, you have to add: SetUpLab.runScratchbotCommands(); at the end.

For example:

    @Override

    public void autonomousInit(){

      Lab.ExerciseEightCode.method1();

      SetUpLab.runScratchbotCommands();

    }

Your task for this exercise is to allow the driver to control the robot during the teleoperated period with the joystick. Then, create an autonomous routine that will move the robot automatically.

Feel free to use your code from previous exercises.
