Command Compositions

A Command Composition is a type of command that is made up of other commands. For example, a score command that is made up of a go to point command, raise arm command, and release piece command. While command compositions are groups of commands, the composition is treated as just one command. There are six main types of command compositions:

  1. Repeating

    Repeats a command over and over

    Finishes only when interrupted

  2. Repeating Sequence

    Repeats a sequence of two or more commands over and over

    Finishes when interrupted

  3. Sequential Command Group

    Each command runs one at a time, in order.

    The command is finished when all of the commands are completed.

  4. Parallel Command Group

    All of the commands run at the same time.

    The command is finished when all of the commands are completed

  5. Parallel Race Group

    All of the commands run at the same time.

    The command is finished when one command finishes

  6. Parallel Deadline Group

    All of the commands run at the same time

    The command is finished when one specific command finishes.

We won’t go over how to make these types of command compositions in these lessons. Right now, it’s more important to understand this conceptually. However, if you’re interested in learning it feel free to read about it on the WPILib Docs.
