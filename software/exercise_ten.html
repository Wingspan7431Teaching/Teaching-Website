Exercise Ten

Task: Create a class called Path that stores a list of points that the robot will travel between. Assume that the movement is being carried out by some other class; the role of this class is solely to keep track of which point in the path the robot is currently on.

The Path Class should contain the following:

  • A variable of type ArrayList<Pose2D> called points
  • A variable of type int called currentPointIdx
  • A constructor of the form Path(ArrayList<double[]> points)
    • Assume each double[] to be of the form {xPosition, yPosition, rotationInRadians}
    • Each constructor should initialize the values of the class variables
  • An overloaded constructor of the form Path(ArrayList<Pose2D> points)
  • An empty constructor containing no parameters
  • A method addPoint() with the parameter double[] point
    • This should add a new point to the points array list.
  • An overloaded method addPoint() with the parameter Pose2D point
  • A method getCurPointIdx() which returns the currentPointIdx
  • A method getCurPoint() which returns the point the bot is currently traversing to. The return type should be Pose2D
  • A method getPathLength() which returns the number of points in the path
  • A method atPoint() which increments the currentPointIdx
    • Should only increment the current position in the path when the next index is less than the length of the list. Use a ternary operator to implement this functionality.
  • A method called isComplete() which returns true or false based on whether the last point in the path has been reached.
