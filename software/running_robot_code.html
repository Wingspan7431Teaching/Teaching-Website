Running Robot Code & Driver Station

To start running your code on the robot, you'll need to connect to the bot via WiFi. Talk to a team member to get connected. Then, open up Driver Station. Driver Station is the app that we use in order to interact with the robot.

The app allows you to see the status of your controllers/ joysticks, switch between modes (such as teleop, auto, and test), and many other important tasks.

A good overview of Driver Station from WPILib is here.

For Scratchbot, if you're on exercise 1 - 7, the runLab() method will be run when you enable test mode. In teleop, there's code that allows you to control the robot using a controller. This will be useful for resetting the robots position.

One last thing, for safety, make sure to yell "Clear!" when you enable the robot. You should speak loud enough for anyone in the area to hear.
