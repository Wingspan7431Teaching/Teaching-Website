Exercise Two

For this exercise, you can create another copy of the Scratchbot repository or delete moveRobot() from the body of runLab and continue working in the same repo.

Task: Create a boolean variable, isTurnClockwise with a value of your choosing. Write a new method randRobot to turn the robot a random number of radians from 0 to 3. If isTurnClockwise is true, turn clockwise. If the variable is false, turn counterclockwise. Print how many degrees the robot turned and in what direction.

Call randRobot in the body of runLab().
