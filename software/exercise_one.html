Exercise One

For this exercise you'll need to fork the Scratchbot repository located here. Fork and clone it in the same way you forked the repo in Exercise Zero. You'll need to use this same template for the following lessons, so it's suggested you give the repository a distinct name such as "Exercise One" rather than "ScratchBot2026."

In the code for scratchbot there’s a few methods you can use to move the robot around:

  • Scratchbot.sb_moveForward(metersToMove)
  • Scratchbot.sb_moveLeft(metersToMove)
  • Scratchbot.sb_moveRight(metersToMove)
  • Scratchbot.sb_turnClockwise(radiansToTurn)
  • Scratchbot.sb_turnCounterClockwise(radiansToTurn)

NOTE: Wingpan ALWAYS use meters for movement and radians for rotation. If you don't know what a radian is, I'd recommend this article.

Task: Navigate to "src\main\java\frc\robot\Lab\Lab.java" inside the repo. Create a method called moveBot() that calls Scratchbot methods to move the robot. Every time the robot finishes a function, add a print statement to the console to state that. When all of the methods are complete, print a statement to say it’s complete. Call moveBot() in the body of runLab()

For example, one might make the robot: drive forward, print "Drove forward 10 meters", turn left, print "Turned pi/2 radians", and finally print "sequence completed."

For instruction on how to run your code on a robot, see Running Robot Code.
