What is Scratchbot?

Scratchbot is similar to Scratch, but you’re writing real code from the get-go! It just has simple pre-made methods that beginners can use without worrying about the complicated stuff behind the scenes. All Scratchbot methods start with Scratchbot.sb_.

Scratchbot is only for learning — by the time you need to code real robot features, you shouldn’t need scratchbot anymore!

You'll be taught the necessary Scrathbot methods in the exercise description.
